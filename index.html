ECE Bop-It Project

Design Project 2: A Day in the Life of an ECE Student

Introduction

For our second design project, our team was tasked with drawing inspiration from the classic childhood game Bop-It and putting our own unique twist on the concept. The final design integrates tasks that simulate common activities and techniques we use daily in our coursework and labs, combining creative problem-solving with technical challenges to provide a fun yet educational experience.

Original Design Concepts

In the early stages, we considered various approaches. For example, we thought of requiring the user to input key presses in a specific order for the Code-It command but opted for a simpler design based on user feedback. Challenges and adjustments are detailed further in the report.

Final Design

Features

  • Multimeter-It: Measure resistance using multimeter probes.
  • Code-It: Press a set of keycaps connected to an Arduino.
  • Solder-It: Touch a soldering iron to a designated pad to complete a connection.

Design Verification

The design underwent incremental prototyping and testing. A breadboard prototype helped validate the circuit and software before transitioning to a PCB. Issues like analog value fluctuations in the multimeter input were resolved with software solutions.

Electronic Design Implementation

The PCB was carefully designed to include all necessary components while adhering to cost, size, and functionality constraints. Key features include capacitors, LEDs, a reset button, and a serial header for debugging.

Software Implementation

The software, written in C, manages user interactions, random command generation, and feedback through audio and visual indicators. Debugging tools like serial output were used extensively during development.

Enclosure Design

The enclosure was designed with portability and usability in mind. Features include a removable panel for battery replacement and a custom layout for components.

Team Contributions

Team members Fredrick Laudati, Leslie McDonough, and Jill Zitcovich collaborated on all aspects of the project, from design and hardware to software development.

Conclusion and Future Work

Future iterations could include a clear acrylic enclosure, additional resistors for teaching color codes, and a volume control knob.
